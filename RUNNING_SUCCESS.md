## 💻 How to Stop

When you're done:
1. Press **Ctrl + C** in each Command Prompt window
2. Type **Y** and press Enter
3. Close the windows

---

## 🛠️ Troubleshooting

### If something doesn't work:

1. **Check all 3 windows are running:**
   - MongoDB (mongod)
   - Backend (npm run dev)
   - Frontend (npm run dev)

2. **Restart if needed:**
   - Stop with Ctrl+C
   - Start again following the steps

3. **Check for errors:**
   - Look for red text in Command Prompt
   - Check browser console (F12)
