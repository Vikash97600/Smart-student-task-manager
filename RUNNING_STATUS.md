## 🚀 Quick Start

### Access the Application
Open your browser and navigate to: **http://localhost:3000**

### First Steps
1. Click **"Register"** in the top-right corner
2. Create your account:
   - Name: Your Name
   - Email: your.email@example.com
   - Password: At least 6 characters
3. Click **"Create Account"**
4. You're automatically logged in to the Dashboard!

### What You Can Do Now

📊 **Dashboard**
- View your productivity statistics
- See total tasks, completed, pending, and overdue
- Check your weekly progress chart
- Monitor your productivity percentage

📝 **Task Management**
- Click **"Add New Task"** to create tasks
- Set title, description, subject, due date, priority
- View all tasks in the Task List
- Search and filter tasks
- Mark tasks as completed
- Edit or delete tasks

📅 **Calendar View**
- Navigate to the Calendar tab
- See all your tasks on a monthly calendar
- Blue dates = tasks due
- Green dates = tasks completed
- Red dates = overdue tasks
- Click any date to see tasks for that day

## 📝 API Endpoints (Live)

All endpoints are accessible at: http://localhost:5000/api

- `POST /api/auth/register` - Register user
- `POST /api/auth/login` - Login
- `POST /api/auth/logout` - Logout
- `GET /api/tasks` - Get all tasks
- `POST /api/tasks` - Create task
- `GET /api/tasks/:id` - Get single task
- `PUT /api/tasks/:id` - Update task
- `DELETE /api/tasks/:id` - Delete task
- `GET /api/tasks/dashboard/stats` - Get statistics
